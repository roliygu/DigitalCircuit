DigitalCircuit
==============
这个project是实现一些数字电路计算的poject，目前计划的功能有：
1.输入逻辑式，生成真值表；
2.输入两个逻辑式，判断是否等价：
	生成两个逻辑式的真值表，比较是否完全相同
3.输入逻辑式，生成最小项或最大项（本质就是生成合取析取范式）：
	还是要利用生成的真值表再做解析
4.输入逻辑式：
	1）返回化简结果：
		卡洛图求解法，感觉比较难；
	2）返回有原件限制的化简结果：
		比如要求只用与非门之类的
	额外功能：统计最简电路使用到的电路元件种类和个数
附录：
	生成真值表：
		解析逻辑式中出现的字母种类，生成对应的0，1全排列；
		利用栈和前缀表达式将真假值带入逻辑式，求逻辑结果。带入过程如流程图1-1。(edx是亿图图示软件的文件格式）
